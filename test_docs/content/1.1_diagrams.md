# Diagramme und Visualisierung

In diesem Abschnitt werden die Diagramm-Fähigkeiten des PDF-Generators demonstriert. Wir nutzen Mermaid.js für die Erstellung von Diagrammen direkt aus dem Markdown.

## Flussdiagramm

Hier ist ein einfaches Flussdiagramm, das den Prozess der PDF-Erstellung zeigt:

```mermaid {PDF Erstellungsprozess}
graph TD
    A[Markdown Dateien] --> B[Markdown Parser]
    B --> C[Internal Block Model]
    C --> D[PDF Engine]
    D --> E[E.ON Brix Sans PDF]
```

## Sequenzdiagramm

Ein Beispiel für ein Sequenzdiagramm:

```mermaid
sequenceDiagram
    participant User
    participant CLI
    participant Engine
    User->>CLI: docgen build
    CLI->>Engine: Parse Config & Content
    Engine-->>CLI: Blocks
    CLI->>Engine: Render PDF
    Engine-->>User: Test.pdf
```
