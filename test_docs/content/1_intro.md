# Einführung in DocGen

Willkommen zur Dokumentation von **DocGen**, dem professionellen PDF-Generator.

## Design-Prinzipien

Dieses Dokument wurde mit folgenden Vorgaben erstellt:
- **Schriftart**: Arial (Regular, Bold, Italic)
- **Code-Darstellung**: Monospace-Schriftart, kursiv, im **Light Mode**
- **Container**: Code-Blöcke sind in grauen Containern eingefasst

## Code-Beispiel (Light Mode)

Hier ist ein Beispiel für die Code-Darstellung:

```go
package main

import "fmt"

// Main Funktion
func main() {
    message := "Hallo E.ON Welt"
    fmt.Println(message)
}
```

Die Syntax-Hervorhebung nutzt das `github` Theme für eine klare Lesbarkeit im hellen Dokumenten-Layout.
